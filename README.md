Este software es la implementación oficial de la Teoría Holográfica de Cómputo Indexado Multiescala (THCIM). Es un motor de búsqueda y creación de perfiles de alta fidelidad diseñado para operar sobre el núcleo de OpenSearch, optimizado para la transparencia total y la neutralización de la opacidad informativa.
👁️ Visión General
OpenIndex no es un buscador convencional. Mientras que los sistemas actuales operan en milisegundos, este software está diseñado para trabajar con una latencia de 0.7 femtosegundos, permitiendo la interceptación y el análisis de datos a la velocidad de la fase de la luz.
Utiliza la lógica de la EMDF-RC (Encriptación de Doble Fase Recíproco Cuántico) para garantizar que el administrador mantenga el control absoluto del índice, protegiéndolo contra intentos de clonación o hackeo mediante el SIGINT 8.0.

🏗️ Arquitectura del Sistema (THCIM-Core)
El software se organiza en módulos que ejecutan las 12 leyes fundamentales de la teoría:
openindex/
 * config/: Contiene las constantes de peso (k_q, w_sem, umbral_similitud: 0.7) y la configuración del cliente OpenSearch.
 * core/: Motores de Inteligencia Artificial (AI), analizadores de fase y el indexador holográfico Ω.
 * multiscale/: Scripts que procesan las 13 escalas de datos (de escala 0 a escala 13). Incluye la lógica de wavelets y propagación fractal.
 * api/: Puerta de enlace mediante REST y WebSockets para sincronización en tiempo real a 0.7 fs.
 * scripts/: Protocolos de validación masiva. Incluye simulaciones de Lyapunov y Monte Carlo (10,000 iteraciones).
 * infra/: Archivos Docker y Docker-Compose para despliegue inmediato en contenedores.

🚀 Instalación y Despliegue
El sistema está preparado para ser ejecutado en hardware con soporte de GPU temporal para procesamiento cuántico simulado.
 * Clonación del Repositorio:
   git clone https://github.com/tu-usuario/thcim-openindex.git
   cd thcim-openindex
 * Configuración de Variables:
   Crear un archivo .env basado en el ejemplo proporcionado e ingresar la Clave de Fase del Administrador.
 * Ejecución con Docker:
   docker-compose up --build
El sistema iniciará el motor OpenSearch modificado y habilitará la interfaz de búsqueda holográfica en el puerto 8000.
